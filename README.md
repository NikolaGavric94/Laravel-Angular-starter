#Resursi

Da bi pokrenuli projekat potrebno je da uradite sledeće iz terminala/command prompt-a:
```javascript
cd <putanja ka projektu>
composer install
cd public/
bower install
npm install
cd ../
```
Posle ovih komandi potrebno je da se isprati Laravel tutorial za instaliranje svih resursa za dizanje Homestead (lokalnog servera).
Koji se nalazi na ovom repozitorijumu u grani "laravel". Kao poslednja komanda treba da bude `vagrant up` i ta komanda treba da vam podigne server i onda kucajte u browseru http://homestead.dev i trebalo bi da vam se prikaze stranica,
ukoliko to nije slučaj, potrebno je da pogledate koji ste korak preskočili.
